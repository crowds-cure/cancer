# crowds-cure / cancer

### Setup and Run
- npm install
- npm start

### Build
To build the application you can run this command in the root filder:

```
$ npm run build
```

This will create the `bundle.js` and the `style.css` files.

This will start a local server that communicates with a live Couch DB.
